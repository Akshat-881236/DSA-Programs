CSS ( Cascading Style Sheets )

Definition :

CSS is a style sheet language used to control the presentation of HTML elements on a web page. CSS allows separation of content and design, improving maintainability and reusability. CSS controls layout, colors, fonts, spacing, responsiveness and visual hierarchy. CSS works using selectors, properties and values applied through cascading and inheritance rules.
