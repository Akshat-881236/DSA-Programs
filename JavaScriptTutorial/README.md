## Variable in JS

<p>Variables are containers for data .</p>

## Variable Rules

<li> Variables names are case sensitive ; "a" & "A" is different .</li>
<li> Only letters , digits , underscore (_) and $ is allowed. (not even space)</li>
<li> Only a letter, underscore (_) or $ should be 1st character.</li>
<li>Reserved words cannot be variables names.</li>

## let , const & var

<h3>var : </h1>
<p>Variable can be re-declared & updated . A global scope variable.

<h3>let : </h3>
<p>Variable cannot be re-declared but can be updated . A block scope  variable .</p>

<h3>const :</h3>
<p>Variable cannot be re-declared or updated . A block scope variable .</p>
